
# HR Analytics Dashboard

This repository contains a Power BI dashboard designed to analyze employee attrition trends and support HR decision-making with data-driven insights.

## Dashboard Features

- Key Metrics:
  - Total Employees: 967
  - Attrition Count: 133
  - Attrition Rate: 14%
  - Total Age, Monthly Income, and Years at Company

- Visualizations:
  - Attrition by Education
  - Attrition by Age Group
  - Attrition by Gender
  - Attrition by Salary Range
  - Attrition by Job Role
  - Attrition Trend by Years at Company

## Use Cases

- Identify departments or roles with high employee turnover
- Analyze demographic and compensation factors influencing attrition
- Develop targeted employee retention strategies

## Tools Used

- Power BI
- Sample HR dataset
